## F1 — Core analysis (Roslyn)

- [ ] F1.2 Kumpulin method symbols project-owned (`IMethodSymbol`) + posisi source (`file:line`) buat editorURL.
- [ ] F1.3 Call graph: per method body, walk `InvocationExpressionSyntax` → `SemanticModel.GetSymbolInfo` → callee `IMethodSymbol`. Edge caller→callee (project-owned only).
- [ ] F1.4 Interface dispatch (≈CHA): `SymbolFinder.FindImplementationsAsync` buat resolve call lewat interface ke impl konkret.

- [ ] F1.6 **Auto-layer** (port BFS chain inference dari endpoint: entry=controller, calls-onward=service, sink=repository). Default ON.
- [ ] F1.7 Builder: nodes + call edges + collapse-helper + prune isolated + sort. Output graph.json. Verify match shape Go.
- [ ] F1.8 Skip generated: file `// <auto-generated>` header atau `.g.cs`/`.Designer.cs` → exclude (analog Go marker).
